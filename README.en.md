# LibraPlatform

#### Description
目前Libra-Platform平台还处于开发阶段，计划于第四季度中上旬开源，先Star不迷路。基于SpringCloud(2020.0.0) + SpringCloudAlibaba(2021.1) + SpringBoot(2.4.2) + Vue3开发，基于多租户SaaS模式的设计，拥有强大的用户中心，权限基于RBAC设计。Libra-Platform平台架构清晰、代码简介、注解齐全。
